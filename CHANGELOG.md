# Changelog

## 0.1.0 - 2026-02-21
- Initial release.
- Unicode-first slug engine.
- ASCII mode with transliteration and unknown-script hex tokens.
- Semantic mode with optional stopword filtering.
- Emoji and symbol rewrite modes.
- `slugDetailed()` trace output.
- `slugAsync()` optional AI hook with deterministic re-sanitization.
- Presets: `safe`, `cyber`, `unicode`.
- TypeScript types, dual ESM/CJS build, vitest suite, CI workflow.
